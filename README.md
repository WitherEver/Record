 ### 7 - 春天 
 * #### 胆小菇会梦见电子僵王扔冰车吗？
 

     春天：胆小菇会梦见电子僵王扔冰车吗？
     
     大厨：会梦见僵王博士身后跟着大批僵尸
     
     A：会梦见口吐冰球但是没带辣椒
     
     春天：没事儿
     
     春天：这个时候只要给他打开明日方舟，告诉他四管大体力和四管中体力还有四小时过期。
     
     然后他的脸庞会逐渐红润，片刻后身体迸发出阵阵热浪，这热浪有形而又无形，赤红的气流如YF-100新一代液氧煤油火箭推进器尾焰怼着脸部直喷，更不要说这热浪还绵延万里有余，僵王的修为，恐怖如斯。
     
     许久，春天发现这热浪的威力会有规律地减弱又陡然增强，又过了段时间，春天脑海中闪过一丝念想：“机会来了”，他从灵戒中取出三枚上品御火符纸，六枚上品急速符纸。
     
     随即在地上飞速绘制着阵法，赶在下一波热浪增强前完成了【大奶大奶带我走阵】
     
     赤红热浪裹挟着僵王，他怒吼一声，整个天空都被一张巨掌遮住，春天仰头望去，发现数以万计的火雨正在倾泻，震惊之余火雨已近在眼前，这时春天才看清，原来那不是什么火雨，那是被赤红热浪加了buff的4060战术核显卡。

     他在古籍上了解过，这4060战术核显卡，出自上古宗门【影未达】乃是宗门绝学。这4060战术核显卡就如同宗门名字一样，影未达，敌已死。
     
     三枚上品御火符构成的防御阵法将近枯竭，春天来不及思考那么多，他将双手放在那对奶子上，如皎月般洁白的奶子，在赤红热浪下蒙上了一层橙红薄纱。
     
     御火阵的能量再度下降，春天的汗液不断渗出，干燥柔顺的手感逐渐被湿滑代替，在阵法破裂之际两道乳白色的光束迸发，随即在阵法中央汇聚，春天眼中白光一闪就去了奶子星。
     
     此时，方圆千里一片火海，元婴期以下皆是灰烬。随之而来的还有代号为5600G的多弹头洲际弹道导弹。据传言所说，5600G搭载的核弹头可在空中依靠高达65W个铭刻阵法迅速分裂为6、12枚同等当量的核弹头。

     其为上古宗门【苏门】的绝学，根据史书记载，5600G由7位牛魔消耗了1.6亿个上品灵石打造而成，出世即巅峰，仅靠一发就让一个拥有4.4万人的邪恶宗门归顺，当年苏门新生招收再创新高，高达3.9万名。
